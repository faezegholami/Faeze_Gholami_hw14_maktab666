تفاوت npm و npx
کلید واژه npm مخفف عبارت node package manager میباشد و برای نصب وابستگی های لایبری به کار میرود. این کلیدواژه پکیج را به صورت سراسری نصب میکند و در دراز مدت فضا اشغال میکند.
کلید واژه npx مخفف عبارت node package execute میباشد که ابتدا بررسی میکند لایبری موردنظر نصب شده است یا خیر سپس اقدام به اجرای آن می نماید. این کلیدواژه سراسری نصب نمیکند و نیاز به نگرانی برای اشغال فضا نیست.
npx از ورژن 5.2.0 به بعد اضافه شده است و کار را برای برنامه نویسان به عبارتی ساده تر کرده است.
